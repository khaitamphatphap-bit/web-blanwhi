# B&L Studio Minimal Fashion Shop

Prototype frontend cho shop thời trang tối giản, cao cấp, ít sản phẩm.

## Tech

- Next.js App Router
- React
- Tailwind CSS
- Framer Motion
- Mock product data local array

## Run

```bash
npm install
npm run dev
```

Sau đó mở `http://localhost:3000`.

## Included

- 20 mock products with colors, sizes, stock and media fields
- Quick product listing filters
- Quick add from product card
- Product detail drawer with swatches, size, stock, buy now, combo, voucher hint and video panels
- Cart drawer with quantity editing, voucher suggestions and combo upsell
- One-page checkout
- Voucher engine based on order value and item quantity
- Combo suggestion component
- Detailed size advisor
- Style assistant / FAQ helper
- Custom uniform builder with logo upload, draggable mockup placement, text options and realtime price estimate
